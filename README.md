# MAJUU V2 Foundation

This repository contains the MAJUU V2 starter foundation built with:

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui-inspired component structure
- Lucide Icons
- Framer Motion
- Supabase (Auth, Storage, Realtime)
- TanStack Query
- React Hook Form + Zod
- Recharts
- Capacitor-ready project structure

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY`, and `SUPABASE_SERVICE_ROLE_KEY`.
3. Install dependencies with `npm install`.
4. Run the dev server with `npm run dev`.
